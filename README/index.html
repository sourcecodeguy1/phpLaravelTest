Php Test Installing

Installing Project.

  1. After unzip the download zip file you will get /phpLaravelTest file, These are the files that you need.
  2. You have to upload this on your server or direct unzip this on your server. If you are using Xammp or other local server you have to unzip it on /htdocs folder for Xampp or on other folder for that speccific server
  3. Now go to your database and create a new database and name it phplaraveltest or as you want. There is a file on /phpLaravelTest/phplaraveltest.sql, this is the database file. You have to import this to your database more about database upload...
  4. After Importing into database you have to go to /phpLaravelTest/.env file. You have to give the proper database name, username and password more about env file...

Upload on database

  1. Go to your database or if you are in xampp or other local server go to /phpmyadmin
  2. Go to that database that you have created. Here we created a database name /phplaraveltest
  3. Now, go to Import and choose /phpLaravelTest/phplaraveltest.sql this file. And click on go.
  4. Now, have the database ready

Fix .env file

  1. To fix env file first go to /phpLaravelTest/.env. And open it on sublime text or any other html editor. You can also do it on Notepad.
  2. Now set the database name, username and password. For demo our database name is phpLaravelTest, our username is root and Password is 123123. These are demo please input the proper name, username and password
